3D Chess & Go

experimental 3D board games • basic Three.js implementation

Two classic games visualized in 3D space. Camera can be rotated/zoomed, pieces are simple cubes, and the logic covers standard rules (with some limitations). Network play works if you have a WebSocket server.

♜ ♞ ♝

Chess

Standard chess rules, 3D board (6x6x8 layout). Move highlighting, check detection, undo, save/load. No AI – human vs human or network.

✓ 3D camera (rotate/zoom) ✓ Network rooms (with chat) ✓ Custom colors / board appearance ✓ Undo & move history

Go

Go rules on a 3D grid (adjustable size, default 6x6x8). Territory scoring, komi, pass/resign, capture detection, and network multiplayer.

🌐 Network

Change board colors, piece colors, background. Simple but adjustable.
